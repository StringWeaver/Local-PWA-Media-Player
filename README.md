# Local PWA Media Player

[**Live Demo**](https://stringweaver.github.io/Local-PWA-Media-Player/)

A fast, minimalist local media player that runs entirely within your browser.

## Features

- **Format Support:** Easily open and play **MKV** and **MP4** video files.
- **Safari Compatible:** Fully supports the Safari browser.
- **Subtitle Support:** 
  - Automatically loads **embedded (internal) subtitles** from your video files.
  - Supports importing external **ASS** and **SRT** subtitle files manually.
- **Hardware Decoding & HDR:** Powered by the browser's native video player, ensuring perfect hardware decoding and full HDR support without battery drain.
- **Progress Saving:** Automatically remembers your playback progress so you can pick up exactly where you left off.

---

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
